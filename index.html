How to apply CSS to HTML

You can include an external CSS file in an HTML document by linking it in the head section. Here's an example: 

<link rel="stylesheet" href="styles.css">
This will link the file `styles.css` to your HTML document, applying the styles defined within that file to your webpage.

How to use JavaScript in a webpage

To link an external JavaScript file to your webpage, use the script tag in the head or at the end of the body section. Here's an example: 

<script src="script.js"></script>
You can also write JavaScript directly within your HTML file using the <script> tag. For example: 
<script>
console.log("Hello, world!");
</script>
